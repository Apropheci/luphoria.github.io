Bookmarklet Hub v0.1 [alpha build 1]

Bookmarklet Hub is a collection of useful bookmarklets. These bookmarklets only depend on this .github.io page, so if you have an internet filter as long as this page is allowed you'll be fine.

Currently includes:

- Firebug Lite [Mozilla]

- UBer version 2.8 [luphoria.github.io]

- CCursor v0.1 [luphoria.github.io]

- Edit This Page [simple script, created by nobody]

- Finish Editing this Page [simple script]

- X-Ray Goggles [Mozilla]

- Clean Stream [cwmonkey]

- Find + Replace [luphoria.github.io]

- Kick Ass [Erik + Johann Rothoff Anderson]

If you would like to request a bookmarklet to be added, use it inside of the bookmarklet.
